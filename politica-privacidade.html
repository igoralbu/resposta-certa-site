Política de Privacidade

Última atualização: julho de 2026 (v2)

1. Sobre a extensão

Resposta Certa é uma extensão para navegadores Chrome que permite criar e inserir respostas prontas em campos de texto de qualquer site.

2. Dados armazenados localmente

Os dados gerados pelo usuário — templates, categorias e preferências — são salvos exclusivamente no armazenamento local do próprio navegador (chrome.storage.local), no dispositivo do usuário. Esses dados nunca são transmitidos a servidores externos.

3. Dados de uso anônimos (Analytics)

A extensão coleta eventos de uso anônimos via Google Analytics 4 (Measurement Protocol) para fins de melhoria do produto. Nenhum dado pessoal ou conteúdo de templates é enviado. Os eventos coletados são:

  • Instalação da extensão
  • Criação de templates
  • Inserção de templates (incluindo a plataforma — ex: WhatsApp Web, Gmail)
  • Escolha de nicho no onboarding
  • Atingimento de limites do plano gratuito

Cada instalação recebe um identificador aleatório (client_id), gerado localmente e sem vínculo com dados pessoais.

4. Dados que NÃO coletamos

  • Dados pessoais de identificação (exceto o e-mail fornecido voluntariamente ao solicitar a Versão Pro — veja a seção 4.1)
  • Conteúdo dos campos de texto onde a extensão é utilizada
  • Conteúdo dos templates criados pelo usuário
  • Histórico de navegação

4.1 Lista de interesse na Versão Pro

Na tela de Configurações, o botão "Solicitar Versão Pro" abre um formulário opcional onde o usuário pode digitar seu e-mail para ser avisado quando a Versão Pro estiver disponível. Esse envio só acontece se o usuário preencher o campo e clicar em enviar — nunca automaticamente. O e-mail informado é enviado a um formulário Google (Google LLC), que o armazena numa planilha privada de acesso restrito ao desenvolvedor da extensão. Ele é usado exclusivamente para avisar sobre o lançamento da Versão Pro e não é vinculado a templates, uso da extensão ou aos eventos anônimos de analytics descritos na seção 3.

5. Permissões utilizadas

  • storage — para salvar templates e preferências localmente no navegador
  • clipboardWrite / clipboardRead — usadas internamente para inserir texto em editores avançados (como WhatsApp Web), que exigem colar via área de transferência. Antes de colar o template, a extensão lê o conteúdo atual da área de transferência apenas para devolvê-lo logo em seguida, evitando apagar algo que o usuário tinha copiado. Esse conteúdo passa apenas pela memória do navegador — nunca é armazenado nem enviado a qualquer servidor
  • Script injetado em todas as páginas — para detectar campos de texto e exibir o menu de templates ao digitar "/" em qualquer site

6. Compartilhamento de dados

Nenhum dado identificável é compartilhado com terceiros, com uma exceção: se o usuário optar por solicitar a Versão Pro informando seu e-mail (seção 4.1), esse e-mail é enviado ao Google Forms. Os eventos de analytics são enviados ao Google Analytics 4 de forma anônima, conforme descrito na seção 3.
